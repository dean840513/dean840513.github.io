BIOS安装教程

注意：安装系统需要一定的计算机知识，操作不当可能会导致计算机上的数据永久丢失！！！继续前请一定先确保好您已了解了相关知识和相应的风险，并且建议您如果有重要资料请一定要先做好备份。
1. 下载iso文件到您的电脑。下载地址：
天翼云盘：https://cloud.189.cn/t/fIbQ7rqyA7Bj（访问码：zws1）
Google Drive: https://drive.google.com/drive/folders/16Kc6FGvx9gLv-VU7axWit7B3nSCJZlxa?usp=sharing
2. 准备一个2G或以上U盘，确保其中没有您需要的资料，因为之后的操作会清空此U盘。
3. 使用工具将iso文件写入U盘，推荐rufus或ultraiso。如不知道怎么做的，请搜索制作ubuntu启动u盘的教程，方法是一样的。注意这一步会清空您的U盘，请确保里面没有您需要的资料。
4. 使用U盘启动计算机。各品牌计算机使用U盘启动的方法各不相同，以Dell为例，是按F12进入启动菜单。其他品牌请自行搜索方法。
5. U盘成功启动的话，会自动启动安装程序，安装程序与ubuntu完全相同，建议先了解一下ubuntu安装一些相关知识。可以参考以下的选项：
语言：中文
键盘布局：汉语
无线网络：不用连
更新选项：最小安装，安装时下载更新不选
安装类型：与Windows共存
时区：Shanghai
创建用户：用户名自己起，密码一定要记住。不想每次输入密码的话，可以把自动登陆先上
6. 然后就等待安装完成后重启电脑，没什么问题的话，就能进入系统了。

BIOS installation steps

Attention: Please make sure you know what you are doing before continue. It is recommended you learn some linux knowledge and backup your important files before installation.
1. Download ios file. Download Link:
Google Drive: https://drive.google.com/drive/folders/16Kc6FGvx9gLv-VU7axWit7B3nSCJZlxa?usp=sharing
天翼云盘：https://cloud.189.cn/t/fIbQ7rqyA7Bj（访问码：zws1）
2. Prepare a 2G or more USB Disk。Make sure nothing you need in this disk. The disk will be empty in step 2.
3. Use a tool to write iso file to your USB Disk. It is recommended use rufus or ultraiso. If you don't know how to do, please Google how to create a ubuntu bootable usb. Please note all data in your USB Disk will be erased.
4. Use USB Disk to boot the computer.
5. After USB Disk boot, the installation program will start. It is same as Ubuntu's installation program. If you don't know what's the meaning, Google is a good beginning. I also give some recommend options to you:
Language：as you wish
Keyboard：as you wish
WIFI：No need
Update option：minimal installation
Download updates while installing: No need
Installation type: alongside with Windows.
Username & Password: as you with but please remember your password. It is important.
6. Reboot. If everything is fine, you can use your new system now. Good luck.
