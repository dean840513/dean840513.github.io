EFI安装教程

注意：安装系统需要一定的计算机知识，操作不当可能会导致计算机上的数据永久丢失！！！继续前请一定先确保好您已了解了相关知识和相应的风险，并且建议您如果有重要资料请一定要先做好备份。
0. 下载iso文件到您的电脑。下载地址：
天翼云盘：https://cloud.189.cn/t/Vjm6Rr2iABnq（访问码：arp8）
GitHub: https://github.com/dean840513/haoos/releases
1. 准备一个剩余空间2G以上的U盘。
2. 关闭Secure Boot。
3. 使用您熟悉的工具将磁盘划分出一块预留的空间，建议至少30G。如果有windows7及以上系统的话，建议使用windows磁盘管理工具的"压缩卷"功能来操作。如不知道怎么做的，请自行搜索"windows磁盘管理压缩卷"，来查找相应教程。
4. 装载或解压iso文件，将其中的所有内容全部复制到U盘根目录下。
5. 使用U盘启动计算机。各品牌计算机使用U盘启动的方法各不相同，以Dell为例，是按F12进入启动菜单。其他品牌请自行搜索方法。
6. U盘成功启动的话，会自动启动安装程序，安装程序与ubuntu完全相同，建议先了解一下ubuntu安装一些相关知识。可以参考以下的选项：
语言：中文
键盘布局：汉语
无线网络：不用连
更新选项：最小安装，安装时下载更新不选
安装类型：与Windows共存（注意：这步一定要按照自己实际情况来，操作不当会导致无法开机甚至是数据丢失！！！）
时区：Shanghai
创建用户：用户名自己起，密码一定要记住。不想每次输入密码的话，可以把自动登陆先上
7. 然后就等待安装完成后重启电脑，没什么问题的话，就能进入系统了。

EFI installation steps

Attention: Please make sure you know what you are doing before continue. It is recommended you learn some linux knowledge and backup your important files before installation.
1. Prepare a 2G USB Disk。
2. Close Secure Boot。
3. Make a 30G free partition. If you have windows 7 or windows 10, you can use windows disk manager to do this. If you don't know how to do, Google is a good beginning.
4. Copy all contents which in iso file to USB Disk's root directory.
5. Use USB Disk to boot the computer.
6. After USB Disk boot, the installation program will start. It is same as Ubuntu's installation program. If you don't know what's the meaning, please use Google. I also give some recommend options to you:
Language：as you wish
Keyboard：as you wish
WIFI：No need
Update option：minimal installation
Download updates while installing: No need
Installation type: alongside with Windows. Please note this step is very important. Please refer your real situation to do this step. Or your computer maybe cannot boot or lost some data.
Username & Password: as you with but please remember your password. It is important.
